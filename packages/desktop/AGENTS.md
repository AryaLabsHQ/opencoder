# Desktop Application

**Package:** `packages/desktop`
**Type:** Electron desktop app

## Overview

Desktop application wrapping the core TUI. Built with Electron + Tauri (src-tauri/).

## Structure

```
packages/desktop/
├── src/            # Electron frontend
└── src-tauri/      # Tauri native bindings
```

## Commands

```bash
# Dev
bun dev

# Build
bun build
```
